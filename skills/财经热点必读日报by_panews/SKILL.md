# 财经热点必读日报（专业版）

基于 PANews 每日必读，生成专业财经热点分析报告。

## 核心升级（v2.0）

| v1.0 | v2.0 |
|------|------|
| 罗列标题 | 一句话总结核心观点 |
| 不获取正文 | 调用 get-article 获取正文分析 |
| 简单列表 | 情绪分析+今日焦点+分类板块+必读+建议 |
| 无分类 | ETF/机构、监管政策、技术/生态、市场/行情 |
| 无 | 市场情绪判断 + 一句话操作建议 |

## 工作流程

### Step 1: 获取 PANews 每日必读

```bash
# 获取每日必读列表
curl -s "https://api.panewslab.com/v3/articles?cid=8&limit=20"
```

### Step 2: 获取文章正文（新增）

对每篇重点文章调用：
```bash
# 获取文章详情
curl -s "https://api.panewslab.com/v3/article/{article_id}"
```

## 注意事项

1. 必读内容：一句话总结比标题更重要
2. 分类要准确，避免跨类
3. 情绪分析要客观，操作建议要保守
4. 每天至少 3 条必读推荐

## API 依赖

- PANews API (https://api.panewslab.com)
- get-article.mjs (获取正文)
